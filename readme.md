Contexto

Pequenos comerciantes frequentemente enfrentam dificuldades para organizar seus pedidos, calcular valores com desconto corretamente, validar informações de clientes e manter um catálogo atualizado de produtos.

Esta aplicação foi desenvolvida como a primeira etapa de um sistema simples de gestão de vendas, com foco na aplicação prática de conceitos fundamentais de JavaScript no front-end.

---

##  Objetivo da Atividade

Desenvolver as funcionalidades iniciais de um sistema de vendas, incluindo:

- Estruturação do projeto em JavaScript  
- Criação de um catálogo de produtos  
- Cálculo básico de vendas e descontos  
- Consumo de API externa (consulta de CEP)  
- Exibição de informações no front-end  

##  Estrutura do Projeto

index.html - Estrutura da aplicação
style.css - Estilização da interface
script.js - Lógica da aplicação (JavaScript)
README.md - Documentação do projeto

##  Como Rodar o Projeto

###  Opção 1 – Abrindo diretamente no navegador

1. Baixe ou clone o repositório.  
2. Abra o arquivo `index.html` no navegador.  


###  Opção 2 – Usando Live Server (Recomendado)

1. Instale a extensão **Live Server** no VS Code.  
2. Clique com o botão direito no arquivo `index.html`.  
3. Selecione **Open with Live Server**.
